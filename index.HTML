Face Identification Project

Project Overview

This project focuses on face identification using various techniques such as Histogram of Oriented Gradients (HoG), Local Binary Patterns (LBP), and Convolutional Neural Networks (CNN). The goal is to develop a model that can accurately identify individuals from their facial images.
