Project: Face Identification

CSL2050 Group 8

Project Overview

This project aims to classify face images into different classes by extracting and comparing various feature representation techniques. The problem statement is to identify a face image by assigning it to one of the classes(i.e thier names) using Local Binary Patterns (LBP), Histogram of Oriented Gradients (HoG), and Convolutional Neural Network (CNN) features. The Labeled Faces in the Wild (LFW) Deep Funneled Dataset, is utilized for this task. The dataset is preprocessed by extracting image paths and loading individual images with constraints i.e. a minimum number of images per person, image slicing, resizing, and color conversion. The implemented methadalogy involves data extraction and preprocessing, feature extraction using LBP, HoG, and CNN techniques, dimensionality reduction using PCA and LDA, and implementing various classifcication models such as Naive Bayes, K-Nearest Neighbors, Random Forest Classifier, Support Vector Classifier, and Artificial Neural Network. The report presents the accuracies achieved by each model after training on the transformed dataset obtained after dimensionality reduction using PCA and LDA,in detail.

Watch the project video on Youtube.

Conclusion:

The project successfully demonstrates the application of different feature extraction techniques and machine learning models for face image classification. The comparison of various models provides insights into the effectiveness of each approach for this specific task.
